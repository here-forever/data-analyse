# Frontend React Skeleton Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [x]`) syntax for tracking.

**Goal:** Build the frontend React/Vite foundation with routing, app shell, design tokens, API client, query provider, lightweight workspace state, tests, lint, and production build verification.

**Architecture:** The frontend is a React + TypeScript single-page app under `frontend/src`. It uses a small app shell with route composition, feature-neutral UI primitives, TanStack Query for server-state setup, Zustand for local workspace UI state, and a typed API client. This milestone intentionally avoids implementing full product workflows; it creates a tested, extensible skeleton for later MVP pages.

**Tech Stack:** Node.js 24, npm.cmd, Vite, React, TypeScript, React Router, TanStack Query, Zustand, Vitest, Testing Library, ESLint, Tailwind CSS, lucide-react.

---

## Scope

This plan implements Milestone 3 from `docs/MVP_ROADMAP.md`: Frontend Skeleton.

It creates:

- Vite + React + TypeScript project configuration.
- Tailwind CSS and base design tokens.
- React Router setup.
- TanStack Query provider setup.
- Zustand workspace state store.
- Typed API client.
- App shell with left navigation, top status bar, and placeholder workspace home.
- Unit/component tests for shell, API client, and store.
- Frontend documentation.

This milestone does not implement authentication, real dataset pages, charts, dashboards, or backend API integration beyond a typed client helper.

Create `frontend/src/lib/apiClient.ts`:

```typescript
export class ApiError extends Error {
  constructor(
    message: string,
    public readonly status: number,
  ) {
    super(message);
    this.name = "ApiError";
  }
}

type Fetcher = typeof fetch;

export interface ApiClientOptions {
  baseUrl: string;
  fetcher?: Fetcher;
}

export interface ApiClient {
  get<TResponse>(path: string): Promise<TResponse>;
}

function joinUrl(baseUrl: string, path: string): string {
  return `${baseUrl.replace(/\/$/, "")}/${path.replace(/^\//, "")}`;
}

async function readErrorMessage(response: Response): Promise<string> {
  try {
    const payload = (await response.json()) as { error?: { message?: string } };
    return payload.error?.message ?? `Request failed with status ${response.status}`;
  } catch {
    return `Request failed with status ${response.status}`;
  }
}

export function createApiClient({ baseUrl, fetcher = fetch }: ApiClientOptions): ApiClient {
  return {
    async get<TResponse>(path: string): Promise<TResponse> {
      const response = await fetcher(joinUrl(baseUrl, path), {
        headers: { Accept: "application/json" },
        method: "GET",
      });

      if (!response.ok) {
        throw new ApiError(await readErrorMessage(response), response.status);
      }

      return (await response.json()) as TResponse;
    },
  };
}

export const apiClient = createApiClient({
  baseUrl: import.meta.env.VITE_API_BASE_URL ?? "http://127.0.0.1:8000/api",
});
```

  sidebarCollapsed: false,
